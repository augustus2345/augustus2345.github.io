makefile 在 go 语言中使用

借助 Makefile我们在编译过程中不再需要每次手动输入编译的命令和编译的参数，可以极大简化项目编译过程。 make 介绍 make是一个构建自动化工具，会在当前目录下寻找 Makefile或 makefile文件。如果存在相应的文件，它就会依据其中定义好的规则完成构建任务。 Makefile 介绍 我们可以把 Makefile简单理解为它定义了一个项目文件的编译规则。借助 Makefile我们在编译过程中不再需要每次手动输入编译的命令和编译的参数，可以极大简化项目编译过程。同时使用 Makefile也可以在项目中确定具体的编译规则和流程，很多开源项目中都会定义 Makefile文件。 本文不会详细介绍 Makefile的各种规则，只会给出 Go 项目中常用的 Makefile示例。关于 Makefile的详细内容推荐阅读 Makefile 教程。 规则概述 Makefile由多条规则组成，每条规则主要由两个部分组成，分别是依赖的关系和执行的命令。 其结构如下所示： [target] ... : [prerequisites] ... <tab>[command] ... ... 其中： targets：规则的目标 prerequisites：可选的要生成 targets 需要的文件或者是目标。 command：make 需要执行的命令（任意的 shell 命令）。可以有多条命令，每一条命令占一行。 举个例子： build: CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build -o xx 示例 .PHONY: all build run gotool clean help BINARY="bluebell" all: gotool build build: CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build -o ${BINARY} run: @go run ....

《你的灯亮着吗？》书评兼谈我的思考

看了高斯（Donald C.Gause）和温伯格（Gerald.Meinberg）的《你的灯亮着吗？》一书，大脑跟着作者的思路层层解剖，犹抱琵琶半遮面，柳暗花明又一村，原来解决问题可以有很多种不一样的思路，而这些思路拓宽了我想问题的方法。 问题是什么？ 问题是理想状态和现实状态之间的差别。 以前我从未想过，一个问题的出现，意味着什么？问题是理想和现实之间的差别，可不是嘛，一旦理想状态和现实状态出现矛盾和冲突，自己所幻想的和现在之间出现差距，问题就来了。这个解释可谓一针见血。 问题到底是什么？ 每一个解决方案都是新的一个问题。 某些问题最困难的部分就在于发现问题所在。 至少想出问题的三个出错的地方。 借助外国人、盲人、儿童来检验你给出的定义，让自己设身处地到他们的角度上来检验。 每转换一次视角，都会发现新的不协调之处。 即便选了一个新的解决方案，也会随之而来一个新的问题，问题永远存在，只要现实和理想状态之间有差距，问题就会伴随而来。真正解决问题的方案是发现问题所在，要至少发现问题三个以上出错的地方，要借助别人的视线或者角度，去站在平时没有考虑到的角度去思考问题。 解决的问题是什么？ 把问题描述成文字，做文字游戏。 当别人可以自己解决问题时，不要插手。 让解决问题者，亲身体验到问题带来的困扰。 试着把责任归到自己身上。 可以把问题转化为文字或者表格或者统计数字，来运用一些技巧去解决问题；如果一个解决问题的人，并没有面临遇到这个问题的困扰，那就让他亲身体验一下这个问题带来的困扰，这样他才会更有责任感去解决问题。 问题从哪里来的？ 大多数情况下，问题的根源在你自己身上。 要知道自己想要什么。 问题为什么存在？大多数是自己想出来的，自己制造出来的，根源在自己身上，要清楚自己想要什么，问题就好解决了。 解决问题第一步： 不要仓促下结论，也不要忽视第一印象。 别去帮缺乏幽默感的人解决问题。 不要参考别人的解决方法。解决方案不是问题的定义。 即使问题已经解决，也不一定是对的。永远不要停止寻找另一个方案的脚步。 问题的解决方案永远有更好的解决方案，永远不要停止思考的脚步。 如何解决问题？ 反面: 没做好，重新来过。 不知道要什么，后悔。 正面: 对自己要忠实。 接近一个解决方案，考虑道德问题，抛弃感性因素。 对自己要忠实，要知道自己想要什么，找到接近自己想要的那个方案。 理想和现实状态出现差距——问题出现——要知道自己想要什么——设身处地去思考问题——对自己要忠实。

《学会提问》读后感

微信群里，家人总是时不时发来一些养生文章，“80岁大脑不萎缩不痴呆的秘密！50岁以上的人必看”…… 社会热点，各路新闻媒体出动，网络上充斥着各种讨论、争议、观点和所谓的真相…… 影视公众号迫不及待地想告诉我们哪些影视作品不容错过，哪些不看为妙…… 信息时代带来了很多便利，同时也充斥着各种嘈杂，我们被越来越多五花八门的看法、“事实”所包裹，恐慌、焦虑或者迷茫，我究竟该相信谁？ 独立的思考和判断变得愈加重要和宝贵，而批判性思维便是一个实用的利器，它帮助我们在这以贩卖观点以获取流量的时代保留些许自我的空间，给与我们鉴别和筛选信息的能力。 这本书是针对批判性思维的一种提醒，也提供了实用而全面的指导意见。但遗憾的是中文版由于翻译的原因，以及文化背景的差异，读起来不是那么顺畅以及接地气，有些长句显得生硬和别扭。但总体来说仍是一本值得好好学习的方法论书籍，每一章以问题出发，告诫和指导我们如果识别论证中的陷阱，最后还附有思维练习，可以马上学以致用，无论是评价别人的观点还是自己写作所需。 首先，最直接的，判断在讨论什么，是描述性论题（是什么）还是规定性论题（该不该）；结论是什么，理由是什么。挖掘结论和理由背后隐藏的价值观假设（价值取向会影响理由的选择和观点的形成）和描述性假设（有某个隐藏的前提假设会影响理由和结论之间的联系）。 警惕理由中意思不明确的词语、数据的欺骗性以及被选择性忽略的重要信息。这里数据欺骗性是指统计样本是否有代表性、相对数和绝对数的差异、平均数与中位数的差异等，尽管数据有天然的权威性，但很多时候数据可以任意“调整”成结论所需要的样子。 重要的一环是识别推理中是否存在谬误，比如常见的人身攻击谬误、诉诸情感谬误、虚拟两难选择谬误等等（这些具体的定义可以在文章中找到）。建议常常回顾，以提醒我们保持警惕。 证据的效力往往影响结论的可信程度，因此我们需要对所谓的论据保持警惕。 导致事件发生的原因很多时候不是是唯一的，我们可以尝试寻找是否存在其他原因是否存在乙问题的直接解决方案。比如有人认为应该关闭市中心的酒吧，因为自从那些酒吧开业以来有十几个年轻大学生酒精中毒。或许我们可以有其他的解决方式，比如严格执行饮酒年龄的限制。 很多问题并不是非黑即白的，要撇清二分式思维的障碍，添加限定条件（when where why）让你的结论多元化。 希望我能时刻提醒自己并刻意的练习批判性思维，最终形成一种自然的条件反射，不会被轻易误导或是人云亦云；对各种观点包容但不趋同，能够用批判的视角评估所有的断言和看法，尤其是自己的看法。

《自控力》书评以及思考如何控制自己

在这个世界上，我们做了太多的功课去了解别人。我们知道我们的朋友是怎样一个人，他有怎样的性格特征；我们知道我们的父母是怎样一个人，他对一些事物有着怎样的感受；我们甚至知道我们的老师或者领导是怎么一个人，做什么样的事情会让他感到愉悦。 然而，几乎我们很少很少有人去了解自己，自己到底是什么样的人，自己在面对一件事情一个人到底是一种什么样的感受。我们习惯用常识来定义自己，遵循所谓的“正确规则”来规范自己，但往往适得其反。 然而，你连自己是什么样的都不知道，怎么能够增加自控力，把一件事情坚持下去呢。 有一个学心理学的朋友，除了我们平时看到的市面上的心理学书，她们也会跟小白鼠做实验，找人做实验。翻看她的朋友圈，你可以看到很多剖析自己的文章，自己对某一件事情的感受是什么样的，自己心理突然升起的某种情绪是怎么形成的。 但是，现实的我们却把好多情绪藏了起来，以为这个坎翻过去了，事实上这个情绪却会不时的出来，扰乱掉你的一切计划，甚至在某个时刻将你打回原形。而自控力这件事情，更是如此。 我们的心理都住着两个自我，一个是冲动的自我，一个是明智的自我。冲动的自我告诉你要及时行乐，明智的自我告诉你要坚持下去。无时无刻，这两个自我都在做斗争。但是，弄清楚自己的，对你能够在这场斗争中取胜至关重要。 每当我们做出一个决定的时候，我们的脑袋里就会分泌出多巴胺，这种多巴胺会让我们产生兴奋。但事实上，得到那件东西却不会让人感觉到快乐。比如一个坚持减肥的人，当你决定要吃一块巧克力蛋糕的时候，你会感觉很兴奋，但是当你真正吃到那块蛋糕的时候，你却会因为没有坚持减肥而极度愧疚和焦虑。当你告诉自己“自己就是很想吃蛋糕”的时候，可能下次再面对巧克力蛋糕的时候，就没有那么有吸引力了。 而当你心里产生负罪感的时候，你为了消除负罪感，通常做的是吃更多，玩更多。然而，你以为自己会好受一点，但实际上你却更加难受。你的心情并不会按照你想要的方向去发展。明明是你“不想要”的东西，但是你的心里却一遍又一遍的重复那个东西，导致最终屈服于它。 所以，当每次情绪产生波动的时候，记录下自己情绪的变化。决定做一件事情的时候，你到底是快乐幸福还是焦虑的，当你做了这件事情之后，你的情绪发生了什么变化。而当下次你再面对这样的情景时，那个明智的自己很有可能就会获胜。这是最简单也是最实用的了解自己的方法。 我们要做的是，对内要接受自己的感受，对外控制自己的行为。而当你接受了自己的感受，控制行为将变得比以前更加容易。 在《自控力》一书中提出，自控力其实就跟肌肉一样是可以锻炼的，所以了解好自己之后，这些小方法，可能会帮助你提高自己的自控力。 1.冥想 冥想非常有利于训练自己控制自己的思维，而且效果非常明显。冥想的最高境界就是什么都不想，每天花上五分钟，在一个不被打扰的环境，精力完全放在自己的呼吸上。 2.做一些平常不会控制的小事 自控力就跟肌肉训练营一样，做一些平常不会控制的小事，难度小，但是却可以增加自控力的活跃度，更重要的还能让乏味的生活，偶尔有趣。比如，惯用右手的人，使用左手来吃饭；比如，平时说是的地方，强迫自己说是的；比如，穿鞋子你习惯先穿左边，从现在开始先穿右边…… 3.等待十分钟 每次你要做“破功”的事情的时候，等待十分钟。这样一来，我们可以把即时的奖励变成要长期才能拿到的奖励，这件事情对我们的吸引力会增加。另外，等待十分钟再做，就算做了也能降低我们的负罪感，减少我们继续做“破功”的事情。 4.给未来的自己写邮件 我相信，让你设想的未来的自己，一定都是肤白人瘦钱多，还积极向上。所以，每次抑郁的时候给未来的自己写写邮件，发发信息。这样，能让你对现在的状态有清楚的了解，也能用“美好的未来”给自己激励。 5.把“我不要”的事情，变成“我想要” 人就是很奇怪，让你尝试五分钟不要想冰淇淋，但是这五分钟里你除了冰淇淋什么其他的东西都想不到。所以，当你告诉自己不要吃零食，不要打游戏，不要出去浪的时候，头脑里其实在一次又一次的提醒你做这些事情。所以，面临这些情况的时候，应该想我想要什么。我想要吃健康的食物，我想要好好读一本书，我想成为一个牛人…… 6.问问自己到底想要的是什么？ 每次做一个决定的时候，问问自己到底想要的是什么。当你想买冰淇淋的时候，问问自己真正想要的吃冰淇淋，还是要一个魔鬼身材；当你期末复习想玩的时候，问问自己想要的是跟大家一起嗨，还是想要奖学金…… 好像，又回到了“了解自己”。 所以，感受每一次自己的感受，你永远比自己想象的奇妙，你也永远比自己想象的强大。

Error 接口和错误处理
